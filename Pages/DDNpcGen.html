Dungeons and Dragons NPC Generator

Many people running D&D games, or any games for that matter, run into the problem of making NPCs. Whether you have to make a large amount of NPCs, or have to make them on the fly. This is an easy to use generator that works completely offline and is still receiving updates adding more content and features.

The applications contains a lot of data, all of which is stored using the markup language JSON. I learned a lot about markup languages, and how to specifically decompile JSON, and have went on to learn more markup languages such as XML and YAML.
